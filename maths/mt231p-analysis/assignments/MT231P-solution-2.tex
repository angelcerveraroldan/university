    \hBox{
        Let $X$ be the set of all possible words in the English language. Prove that $X$ is countable.
    }

    The set of words in the enlgish language is a finite set. Countable sets need to be either finite, or countably finite. Therefore set of enlgish words is countable.
\end{homeworkProblem}
